Message List, the Project

Message List is a poor man's version of Twitter, used for Assignment 1 of CPSC 436I as an intro project using HTML, CSS, and JavaScript. As you hopefully can see, the website has a list of messages, to which you can add to yourself. You can also clear the list of messages, but new messages added by you are not saved when you exit the page. Messages can be 140 characters or less.
